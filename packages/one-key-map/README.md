A Map-like data structure that only stores the most recently set key/value pair

## Quick Start

```js
import OneKeyMap from '@essentials/one-key-map'

const map = new OneKeyMap()
map.set('foo', 'bar')
map.get('foo')
// bar
map.set('foo2', 'bar2')
map.get('foo')
// undefined
map.get('foo2')
// bar2
```
